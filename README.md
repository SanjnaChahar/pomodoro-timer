# 🍅 Pomodoro Timer

A simple and aesthetic Pomodoro Timer built using HTML, CSS, and JavaScript to help boost focus and productivity.

## 🔧 Features
- 25-minute focus session
- Start, Pause, and Reset buttons
- Clean pastel design with smooth UI
